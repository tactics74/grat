                  Since then, we've changed how we use Python a ton internally. We have

reimplemented all our production service monitoring in Python and also
our production software management infrastructure for a significant

the next-generation thing that we have coming, which is a suite of
programs that will automate the upgrade process for us.

          <div id="intro">
          <!--utf-8-->
          <!--0.7-->
            <p>
              <strong>
              Python is a programming language that lets you work more quickly and
integrate your systems more effectively. You can learn to use Python
and see almost immediate gains in productivity and lower maintenance
costs.
              </strong>
            </p>
            <p>
              Python runs on Windows, Linux/Unix, Mac OS X, and has been ported to
the Java and .NET virtual machines.
            </p>
            <p>
              Python is free to use, even for commercial products, because of its
OSI-approved
              <a class="reference external" href="/psf/license/">
              open source license
              </a>
              .
            </p>
            <p>
              New to Python or choosing between Python 2 and Python 3? Read
              <a class="reference external" href="http://wiki.python.org/moin/Python2orPython3">
              Python 2 or Python 3
              </a>
              .
            </p>
            <p>
              The
              <a class="reference external" href="/psf/">
              Python Software Foundation
              </a>
              holds the
intellectual property rights behind Python, underwrites the
              <a class="reference external" href="http://pycon.org/">
              PyCon
conference
              </a>
              , and funds other projects
in the Python community.
            </p>
            <p>
              <a class="reference external" href="/about/">
              Read more
              </a>
              , -or-
              <a class="reference external" href="/download/">
              download Python now
              </a>
            </p>
